Applies the mapping between the specified value from the Resource\-Record\-Type enumuration and the correct derived class form \hyperlink{class_senergy_1_1_dns_1_1_resource_record_interface}{Resource\-Record\-Interface}. 

If a correct mapping is found, a new instance of the found class is created as a shared pointer, and the specified \hyperlink{class_senergy_1_1_dns_1_1_resource_record_base}{Resource\-Record\-Base} class instance is based in the constructor.

When no mapping could be found, an invalid pointer (null ptr) is returned.
